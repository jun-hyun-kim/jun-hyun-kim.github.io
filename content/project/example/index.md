Course project for a Vision-Language-Model course.

Conditioned human demonstration videos on a Vision-Language-Action (VLA) model to reduce data requirements when finetuning VLA models on downstream embodiments.

- Obtained a 53% success rate gain by conditioning on human videos at inference time.
- Trained the VLA model without cumbersome explicit retargeting methods to improve generality.
